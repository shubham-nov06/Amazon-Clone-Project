# Amazon-Clone-Project

A front-end clone of the Amazon website (learning project).

This project contains HTML, CSS, and JavaScript (and any assets) used to recreate basic Amazon-like UI for practice.

## Contents
- HTML pages
- CSS styles
- JavaScript for interactions
- Assets (images, icons)

## How to run
1. Clone the repo:
   git clone https://github.com/shubham-nov06/Amazon-Clone-Project.git
2. Open `index.html` in your browser or serve with a static server.

## Notes
This is a learning project and not intended for production use.
